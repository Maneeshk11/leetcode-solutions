# Move Zeroes

-----

#### Description: 

Given an integer array ```nums```, move all ```0```'s to the end of it while maintaining the relative order of the non-zero elements.

**Note** that you must do this in-place without making a copy of the array.

#### Examples:

```txt
Input: nums = [0,1,0,3,12]
Output: [1,3,12,0,0]
```

```txt
Input: nums = [0]
Output: [0]
```

#### Constraints: 

- $$1 <= nums.length <= 10^4$$
- $$-2^{31} <= nums[i] <= 2^{31} - 1$$

#### Code: 

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int j = 0, i = 0;
        int n = nums.size()-1;
        while (i < n && j < n) {
            if (nums[i] == 0) {
                while (nums[j] == 0 && j < n) {
                    j++;
                }
                if (nums[j] == 0) {
                    return;
                }
                nums[i] = nums[j];
                nums[j] = 0;
                i++;
            } else {
                i++;
                j++;
            }
        }
        return;
    }
};
```

#### Explanation:

- We will use two pointers, $$i$$ and $$j$$, to iterate over the array.
- We will iterate over the array using the pointer $$i$$ and whenever we find a zero, we will iterate over the array using the pointer $$j$$ to find the next non-zero element.
- We will swap the zero with the non-zero element and continue the process.
- We will continue this process until we reach the end of the array.

#### Complexity Analysis:

- **Time Complexity:** $$O(N)$$ where $$N$$ is the number of elements in the array.
