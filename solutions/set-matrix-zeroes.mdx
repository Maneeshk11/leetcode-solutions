# Set Matrix Zeroes

-----

#### Description: 

Given an ```m x n``` integer matrix ```matrix```, if an element is ```0```, set its entire row and column to ```0```'s.

You must do it **in place**.

#### Examples:

```txt
Input: matrix = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
```

```txt
Input: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
Output: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

#### Constraints: 

- ```m == matrix.length```
- ```n == matrix[0].length```
- ```1 <= m, n <= 200```
- $$-2^{31} <= matrix[i][j] <= 2^{31} - 1$$

#### Code: 

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        set<int>cols, rows;
        int m = matrix.size(), n = matrix[0].size();
        for (int i=0; i<m; i++) {
            for (int j=0; j<n; j++) {
                if (matrix[i][j] == 0) {
                    rows.insert(i);
                    cols.insert(j);
                }
            }
        }
        for (auto x: cols) {
            for (int i=0; i<m; i++) {
                matrix[i][x] = 0;
            }
        }
        for (auto x: rows) {
            for (int i=0; i<n; i++) {
                matrix[x][i] = 0;
            }
        }
        return;
    }
};
```

#### Explanation:

- We can solve this problem by using a set to store the rows and columns that need to be set to zero.
- We iterate over the matrix and whenever we find a zero, we insert the row and column index into the set.
- After that, we iterate over the set and set the corresponding rows and columns to zero.

#### Complexity Analysis:

- The time complexity for this approach is $O(m*n)$.
